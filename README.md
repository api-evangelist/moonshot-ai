# Moonshot AI (moonshot-ai)

Moonshot AI is a Chinese foundation model company best known for Kimi, an LLM with industry-leading long-context capabilities. The Moonshot platform exposes OpenAI-compatible chat completion, files, batch, models, balance, and token-estimation APIs.

**APIs.json:** [https://raw.githubusercontent.com/api-evangelist/moonshot-ai/refs/heads/main/apis.yml](https://raw.githubusercontent.com/api-evangelist/moonshot-ai/refs/heads/main/apis.yml)

## APIs

### Moonshot AI Platform API

OpenAI-compatible inference API for Kimi K2.6, K2.5, K2, and Moonshot V1 models. Supports chat completions (incl. partial mode and tool use), models listing, file upload and content extraction, batch jobs, account balance, and token estimation.

- **Human URL:** [https://platform.kimi.ai/docs](https://platform.kimi.ai/docs)
- **Base URL:** `https://api.moonshot.ai`
